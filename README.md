# b0t

A personal AI companion app for iOS. On-device, markdown-driven, single user, single device.

The user's b0t lives as a directory of plain markdown files in their Documents folder. Apple's Foundation Models framework animates those files. The app is a body for files the user owns.

## What this repository contains

This is the design and content scaffold for b0t v1. Code is generated by Claude Code under the direction of Jamee, working from the documents here.
